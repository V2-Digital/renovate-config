# Renovate Config

Renovate config presets by V2 Digital.

## Usage

Example: `.github/renovate.json`

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "assignees": ["octocat"],
  "extends": ["github>V2-Digital/renovate-config"]
}
```
